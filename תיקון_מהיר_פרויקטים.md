# 🎯 תיקון מהיר - פתיחת פרויקט נופלת

## מה התקלה?
כשפותחים פרויקט, המערכת נופלת עם שגיאה:
```
psycopg2.errors.UndefinedColumn: column l.full_name does not exist
```

## מה התיקון?
עדכנו את השאילתה ב-`server/routes_projects.py` (שורה 248) להשתמש ב-`CONCAT_WS` במקום לגשת ישירות ל-`full_name` שלא קיים.

**לפני:**
```sql
l.full_name  ❌ (העמודה לא קיימת!)
```

**אחרי:**
```sql
COALESCE(CONCAT_WS(' ', l.first_name, l.last_name), l.first_name, l.last_name, '') AS full_name  ✅
```
