# מאגר (Assets Library) - מדריך שימוש

## סקירה כללית

**מאגר** היא תכונה חדשה במערכת המאפשרת לך לנהל ולשתף פריטים, תמונות ומידע עם הבינה המלאכותית בשיחות. הבינה המלאכותית יכולה לחפש במאגר ולשלוח תמונות ללקוחות ב-WhatsApp.

## תכונות עיקריות

### 1. ניהול פריטים במאגר
- ✅ **יצירת פריטים** - הוסף דירות, מוצרים, שירותים או כל סוג פריט עסקי
- ✅ **תיאור מפורט** - הוסף שם, תיאור, קטגוריה ותגיות לכל פריט
- ✅ **גלריית תמונות** - צרף מספר תמונות לכל פריט
- ✅ **שדות מותאמים** - הוסף מידע נוסף ספציפי לעסק שלך

### 2. חיפוש וסינון
- 🔍 **חיפוש חופשי** - חפש לפי שם, תיאור או תגיות
- 🏷️ **סינון לפי קטגוריה** - סדר את הפריטים לפי קטגוריות
- 📊 **סינון לפי סטטוס** - הצג פריטים פעילים או בארכיון

### 3. שימוש בבינה המלאכותית
הבינה המלאכותית יכולה:
- 🤖 **לחפש במאגר** - AI יכול לחפש פריטים רלוונטיים בשיחה עם לקוחות
- 📱 **לשלוח תמונות ב-WhatsApp** - AI יכול לשלוח תמונות מהמאגר ללקוחות
- 💡 **להציע פריטים רלוונטיים** - AI ימליץ על פריטים מתאימים בהתאם לשיחה

## איך להתחיל?

### שלב 1: גישה לעמוד המאגר
1. היכנס למערכת
2. לחץ על **"מאגר"** בתפריט הצדדי (אייקון 📦)
3. תגיע לעמוד המאגר עם רשימת הפריטים

### שלב 2: הוספת פריט ראשון
1. לחץ על כפתור **"פריט חדש"** (למעלה מימין בדסקטופ, או כפתור צף במובייל)
2. מלא את הפרטים:
   - **שם הפריט** (חובה) - שם מזהה לפריט
   - **תיאור** - תיאור מפורט של הפריט
   - **קטגוריה** - למשל: "דירה", "מוצר", "שירות"
   - **תגיות** - מילות מפתח מופרדות בפסיקים
3. לחץ **"צור פריט"**

### שלב 3: הוספת תמונות
1. לחץ על הפריט שיצרת
2. במסך הפרטים, תוכל להוסיף תמונות
3. תמונה ראשונה תהיה תמונת הקאבר
4. אפשר להוסיף עד מספר תמונות לכל פריט

### שלב 4: שימוש עם הבינה המלאכותית
הבינה המלאכותית תשתמש במאגר אוטומטית בשיחות:

**דוגמה לשיחה:**
```
לקוח: "יש לכם דירות בתל אביב?"
AI: [מחפש במאגר]
AI: "כן! יש לנו מספר דירות מעולות בתל אביב. אשלח לך כמה אופציות..."
AI: [שולח תמונות מהמאגר]
```

## שימוש מתקדם

### כלי AI זמינים למפתחים
הבינה המלאכותית משתמשת בכלים הבאים:

1. **assets_search** - חיפוש פריטים במאגר
   ```python
   assets_search(query="דירה בתל אביב", category="דירה", limit=5)
   ```

2. **assets_get** - שליפת פרטי פריט מלא
   ```python
   assets_get(asset_id=123)
   ```

3. **assets_get_media** - שליפת תמונות לשליחה
   ```python
   assets_get_media(asset_id=123)
   ```

4. **whatsapp_send** - שליחת תמונות מהמאגר
   ```python
   whatsapp_send(
       message="הנה כמה אופציות שמתאימות לך",
       attachment_ids=[456, 457, 458]  # תמונות מהמאגר
   )
   ```

## טיפים לשימוש אפקטיבי

### ✅ עשה:
- הוסף תיאורים מפורטים לכל פריט
- השתמש בתגיות רלוונטיות (מיקום, מחיר, סוג וכו')
- עדכן תמונות באיכות גבוהה
- ארגן פריטים בקטגוריות ברורות

### ❌ אל תעשה:
- אל תשאיר פריטים ללא תיאור
- אל תשכח להוסיף תמונת קאבר
- אל תשאיר פריטים לא רלוונטיים בסטטוס "פעיל"
- אל תשכפל פריטים

## שאלות נפוצות

**ש: כמה תמונות אפשר להעלות לכל פריט?**
ת: אין הגבלה טכנית, אבל מומלץ 3-10 תמונות איכותיות לכל פריט.

**ש: הבינה המלאכותית שולחת תמונות אוטומטית?**
ת: כן! כאשר לקוח שואל על פריט, AI יכול לשלוח תמונות רלוונטיות מהמאגר.

**ש: איך מוחקים פריט?**
ת: לחץ על הפריט ובחר "ארכב". הפריט יועבר לארכיון ולא יימחק לצמיתות.

**ש: אפשר לשחזר פריטים מהארכיון?**
ת: כן! שנה את הסינון ל"ארכיון" ועדכן את הסטטוס חזרה ל"פעיל".

**ש: האם הדף מופיע לכל המשתמשים?**
ת: כן! הדף זמין לכל המשתמשים (סוכנים, מנהלים ובעלים) עם הרשאות מתאימות.
