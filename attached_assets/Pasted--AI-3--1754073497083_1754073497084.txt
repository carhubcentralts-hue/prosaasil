מצוין! הנה הגרסה **המעודכנת והמדויקת** להנחיה שאתה שולח לסוכן AI — עכשיו עם הבהרה ברורה:
יש **3 מערכות מופרדות**, אבל לכל עסק/מנהל יש **דף דשבורד ראשי מותאם אישית** שמרכז את כל הגישה, לפי ההרשאות.

---

## ✅ גרסה מעודכנת – טמפלט חדש עם 3 מערכות + דשבורדים מותאמים להרשאות

> 🛑 **שלב ראשון – מחיקה בטוחה בלבד**

**מחק את הטמפלט הוויזואלי הקיים בלבד** (HTML/React ישן, סטייל ישן, תצוגות כלליות).
🛑 **אל תיגע בקוד backend, בשירותים, או בקבצי Flask/Python/Twilio/Whisper/CRM.**

---

## 🧱 בניית טמפלט חדש – Tailwind + React + תצוגה חכמה לפי הרשאות

### ✨ מבנה כללי:

* React + Tailwind
* RTL מלא
* תפריט צד מותאם להרשאות
* מערכת מודולרית – כל מערכת נפרדת (CRM / WhatsApp / Calls)

---

## 🏠 דף ראשי לעסק = `/dashboard`

* **נראה רק מה שהעסק מורשה להשתמש בו**
* כפתורים נכנסים ל־:

  * 📞 מוקד שיחות (`/calls`)
  * 💬 וואטסאפ (`/whatsapp`)
  * 📁 CRM (`/crm`)
* תצוגה רספונסיבית גם במובייל

### דוגמה:

```json
{
  "calls_enabled": true,
  "whatsapp_enabled": false,
  "crm_enabled": true
}
```

⬅️ יראה כפתורים רק ל־CRM ו־Calls.

---

## 👑 דשבורד למנהל מערכת = `/admin/dashboard`

* מציג **רשימת עסקים**
* עבור כל עסק:

  * צפייה בהרשאות קיימות
  * שינוי הרשאות (checkbox)
  * כפתור “כניסה כעסק” → פותח את `/dashboard` של אותו עסק
  * סיכום פעילות (מספר שיחות, הודעות, לקוחות)

---

## 💡 המערכת כוללת 3 מערכות נפרדות לגישה:

| מערכת    | ניתוב       | מי רואה אותה                      |
| -------- | ----------- | --------------------------------- |
| CRM      | `/crm`      | עסקים עם `crm_enabled: true`      |
| WhatsApp | `/whatsapp` | עסקים עם `whatsapp_enabled: true` |
| Calls    | `/calls`    | עסקים עם `calls_enabled: true`    |

ה־Sidebar משתנה בהתאם.

---

## 🧠 פרומפטים חכמים לכל עסק:

* כל עסק מכיל שדה `ai_prompt`
* תשתמש בו בכל פעם שנשלחת הודעה או מתבצעת שיחה
* דוגמה:

```json
"ai_prompt": "אתה נציג מכירות של חברת רכב יוקרה..."
```
