## בדיקת עשן מיד לאחר הפריסה

1. **WS חי** – להתחבר ידנית:
   `wss://<הדומיין שלך>/ws/twilio-media`
   ותראה בלוג: `WS_CONNECTED` / `WS_FRAME`.
2. **Webhooks 200**:

   ```bash
   curl -i https://<host>/webhook/incoming_call
   curl -i -X POST https://<host>/webhook/stream_ended
   curl -i -X POST https://<host>/webhook/call_status
   ```

   אסור לראות 403/501.
3. **Twilio Request Inspector** – בשיחה חדשה לראות 200 על `incoming_call`, ואם הסטרים נסגר – 200 על `stream_ended`.
