🎯 מטרה:
לבנות 3 דפי React מחוברים במלואם ל־backend:
1. `AdminDashboard.jsx`
2. `BusinessDashboard.jsx`
3. `BusinessViewPage.jsx`

כל הנתונים יישלפו מהשרת, כולל סטטוסים, שירותים פעילים, שמות משתמשים, CRM וכו’.

## 📁 קבצים

| דף | מסלול | למי מוצג |
|-----|--------|-------------|
| LoginPage.jsx | `/login` | כל משתמש |
| AdminDashboard.jsx | `/admin/dashboard` | admin בלבד |
| BusinessDashboard.jsx | `/business/dashboard` | business בלבד |
| BusinessViewPage.jsx | `/admin/business/:id/view` | admin בלבד |

## 🛣️ ניתוב והרשאות:

1. לאחר התחברות, שמור ב־`localStorage`:
   - `token`
   - `role`
   - `businessId`

2. ניתוב אוטומטי לפי `role`:
   - `admin` → `/admin/dashboard`
   - `business` → `/business/dashboard`

3. ניתוב דרך `PrivateRoute.jsx`:
   - כל `/admin/*` רק ל־admin
   - כל `/business/*` רק ל־business

## 🧑‍💼 AdminDashboard.jsx

### ✅ שלוף מהשרת:
- `/api/admin/summary` → סטטיסטיקות מערכת כלליות
- `/api/admin/businesses` → רשימת עסקים
- `/api/status` → סטטוס Baileys / Twilio / GPT (למעלה)

### ✅ תצוגה:
- כותרת: “שלום מנהל [שם]”
- סטטיסטיקות כלליות (עסקים, משתמשים)
- כפתורי גישה:
  - 📋 CRM (של המנהל) → `/admin/crm`
  - 💬 WhatsApp (של המנהל) → `/admin/whatsapp`
  - 📞 שיחות (של המוקד הכללי) → `/admin/calls`

### 📋 טבלת עסקים:
לכל עסק:
- שם
- מזהה
- שירותים פעילים (CRM / שיחות / WhatsApp)
- סטטוס GPT / Twilio / Baileys (מ־`/api/status/:business_id`)
- כפתור “שנה סיסמה” → POST `/api/admin/reset-password`
- כפתור “צפה בעסק” → `/admin/business/:id/view`
- כפתור “ערוך עסק”
- כפתור “הוסף עסק” → `/admin/add-business`

## 👀 BusinessViewPage.jsx

- מציג את **כל הדשבורד העסקי של אותו עסק**, כמו שייראה לו
- שליפת נתונים אמיתית מ־`/api/business/:id/*`
- הדף חייב להיראות ולהגיב בדיוק כמו הדשבורד העסקי
- בראש הדף: תווית "מצב תצוגה – מנהל"

> admin יכול לצפות, אבל לא לבצע פעולות בפועל (read-only)

## 🧑‍💻 BusinessDashboard.jsx

### ✅ שלוף מהשרת:
- `/api/business/info` ← פרטי העסק הנוכחי
- `/api/business/services` ← מהם השירותים שהופעלו
- `/api/status` ← סטטוס GPT, Twilio, Baileys
- `/api/business/users` ← משתמשים בעסק

### ✅ תצוגה:
- שלום [שם העסק]
- מזהה עסק, מספר משתמשים
- תוקף חבילה
- תאריך עברי

### 🔐 ניהול סיסמה:
- כפתור “שנה סיסמה” (modal)
- POST `/api/business/change-password`

### 🎛️ שירותים פעילים:
- הצג כפתורים **רק אם השירות מופעל**:
  - 📋 CRM → `/crm`
  - 💬 WhatsApp → `/whatsapp`
  - 📞 שיחות → `/calls`

### 📡 זמינות מערכת:
- GPT (מ־status)
- Twilio
- Baileys
- הצג סטטוסים אמיתיים (🟢/🔴)
